# shopping-lists
App for managing shopping lists, allowing user to:
- add / delete / archive / unarchive shopping lists
- add / delete items from lists

</br><strong>Used libraries:</strong></br>
- Room
- Data Binding
- ViewModel, LiveData
- Navigation
- Hilt
- Coroutines
